# 에이원 과학 전문학원 - 주간 학습 기록 카드 시스템
## 설치 및 사용 가이드

---

## 📋 시스템 구성

이 시스템은 3개의 파일로 구성되어 있습니다:

1. **study-card.html** - 학생/학부모가 보는 학습 기록 카드
2. **admin.html** - 선생님이 사용하는 관리자 페이지
3. **logo.jpg** - 학원 로고 이미지

---

## 🚀 설치 방법

### 1단계: 구글 스프레드시트 준비

#### 1.1 새 스프레드시트 만들기
1. Google Sheets에 접속 (https://sheets.google.com)
2. 새 스프레드시트 만들기
3. 파일 이름을 "에이원 학습기록"으로 변경

#### 1.2 헤더 설정
첫 번째 행(A1~J1)에 다음 내용을 입력하세요:

| A열 | B열 | C열 | D열 | E열 | F열 | G열 | H열 | I열 | J열 |
|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| 학생이름 | 날짜 | 수업명 | 수업내용 | 과제확인 | 과제내용 | 오답체크 | 주간테스트명 | 주간테스트결과 | 수업내용및일정안내 |

#### 1.3 예시 데이터 (2번째 행부터)
```
홍길동 | 2024-11-06 | 물리I | 등가속도 운동, 자유낙하 실험 | 완료 | 교재 p.45-50 문제풀이 | 완료 | 운동학 기본 | 95점 | 다음주는 포물선 운동을 배웁니다. 준비물: 계산기
김영희 | 2024-11-06 | 화학I | 원자의 구조, 전자배치 | 일부완료 | 워크북 1-5번 | 미완료 | 원소의 주기성 | 88점 | 주기율표를 외워오세요.
```

#### 1.4 공유 설정
1. 오른쪽 상단 "공유" 버튼 클릭
2. "링크가 있는 모든 사용자"로 설정
3. 권한: "뷰어" (읽기 전용)
4. "링크 복사" - 나중에 사용할 것이므로 저장해두세요

---

### 2단계: Google Cloud API 설정

#### 2.1 Google Cloud Console 접속
1. https://console.cloud.google.com 접속
2. Google 계정으로 로그인

#### 2.2 프로젝트 생성
1. 상단의 프로젝트 선택 드롭다운 클릭
2. "새 프로젝트" 클릭
3. 프로젝트 이름: "에이원-학습카드"
4. "만들기" 클릭

#### 2.3 Google Sheets API 활성화
1. 좌측 메뉴 → "API 및 서비스" → "라이브러리"
2. "Google Sheets API" 검색
3. "Google Sheets API" 선택
4. "사용" 버튼 클릭

#### 2.4 API 키 생성
1. 좌측 메뉴 → "API 및 서비스" → "사용자 인증 정보"
2. 상단 "+ 사용자 인증 정보 만들기" 클릭
3. "API 키" 선택
4. API 키가 생성되면 "키 복사" - 안전한 곳에 저장!
5. (선택) "키 제한" 클릭 → API 제한 → Google Sheets API만 선택

⚠️ **중요**: API 키는 안전하게 보관하세요!

---

### 3단계: 파일 호스팅

#### 옵션 1: GitHub Pages (무료, 추천)

1. **GitHub 계정 생성** (없다면)
   - https://github.com 접속
   - "Sign up" 클릭

2. **새 저장소 만들기**
   - "New repository" 클릭
   - Repository name: `a1-study-card`
   - Public 선택
   - "Create repository" 클릭

3. **파일 업로드**
   - "uploading an existing file" 클릭
   - study-card.html, admin.html, logo.jpg 드래그
   - "Commit changes" 클릭

4. **GitHub Pages 활성화**
   - Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" 선택
   - "Save" 클릭
   - 5분 후 URL 확인: `https://[username].github.io/a1-study-card/`

#### 옵션 2: 로컬 테스트
- 파일들을 한 폴더에 넣고 `admin.html`을 브라우저로 열기
- 학생들에게 링크를 보낼 수는 없지만 테스트 가능

### 3단계: "설정 저장" 클릭

## 📱 사용 방법

### 학습 기록 작성
1. 구글 스프레드시트에 학생 정보 입력
2. 각 열에 해당하는 정보 작성
3. 과제확인/오답체크: "완료", "미완료", "일부완료" 중 하나 입력
4. 저장 (자동 저장됨)

### 링크 생성 및 전송
1. `admin.html` 열기
2. "링크 생성" 탭 클릭
3. 학생 이름과 날짜 입력
4. "링크 생성" 클릭
5. "링크 복사" 버튼으로 URL 복사
6. 카카오톡으로 학생/학부모에게 전송

### 학습 카드 확인 (학생/학부모)
1. 받은 링크 클릭
2. 학습 내용 확인
3. "인쇄하기" 또는 "PDF 저장" 가능

---

## 💡 팁과 트릭

### 효율적인 데이터 관리
- **날짜 형식**: YYYY-MM-DD (예: 2024-11-06)
- **일관된 학생 이름**: 띄어쓰기나 오타 주의
- **과제/오답 상태**: 정확히 "완료", "미완료", "일부완료" 입력

### 대량 링크 생성
1. "데이터 목록" 탭 사용
2. "데이터 새로고침" 클릭
3. 각 학생별 "링크 복사" 버튼 활용

### 모바일에서 사용
- admin.html은 PC 사용 권장
- study-card.html은 모바일에서도 깔끔하게 표시

### 인쇄 설정
- 브라우저 인쇄 시 "배경 그래픽" 옵션 켜기
- PDF 저장: 프린터 → "PDF로 저장" 선택

## 🔧 문제 해결

### "데이터를 불러올 수 없습니다" 오류
✅ **해결 방법**:
1. 구글 시트가 "링크가 있는 모든 사용자" 공유인지 확인
2. API 키가 올바른지 확인
3. Google Sheets API가 활성화되어 있는지 확인
4. 시트 ID와 시트 이름이 정확한지 확인

### 링크가 작동하지 않음
✅ **해결 방법**:
1. 배포 URL이 올바르게 설정되었는지 확인
2. 파일명이 정확한지 확인 (study-card.html)
3. 브라우저 캐시 삭제 후 재시도

### 로고가 보이지 않음
✅ **해결 방법**:
1. logo.jpg 파일이 같은 폴더에 있는지 확인
2. 파일명이 정확히 "logo.jpg"인지 확인
3. 이미지 파일이 손상되지 않았는지 확인

### API 키 할당량 초과
✅ **해결 방법**:
1. Google Cloud Console → API 할당량 확인
2. 무료 할당량: 하루 100개 요청
3. 필요시 결제 정보 등록 (소액)
